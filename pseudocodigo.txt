Estructura del menú:
1. Se mostrará un título y una lista de opciones numeradas (mínimo 5).
2. El usuario elegirá una opción escribiendo un número.
3. Un bucle repetitivo mantendrá el menú activo hasta que el usuario seleccione "Salir".
4. Cada opción estará controlada por condicionales (SI / SINO SI / SINO).
5. Por ahora, las acciones se representarán con marcadores de posición <acción>, que luego se reemplazarán con la lógica en Python.

Pseudocódigo:

INICIO

    MIENTRAS VERDADERO HACER
        IMPRIMIR "=============================="
        IMPRIMIR "  GESTOR DE NOTAS ACADÉMICAS  "
        IMPRIMIR "=============================="
        IMPRIMIR "1. Registrar nuevo curso y nota"
        IMPRIMIR "2. Mostrar todas las notas"
        IMPRIMIR "3. Calcular promedio general"
        IMPRIMIR "4. Buscar nota de un curso"
        IMPRIMIR "5. Salir"
        IMPRIMIR "Seleccione una opción: "
        LEER opcion

        SI opcion = 1 ENTONCES
            <acción registrar curso y nota>
        SINO SI opcion = 2 ENTONCES
            <acción mostrar todas las notas>
        SINO SI opcion = 3 ENTONCES
            <acción calcular promedio>
        SINO SI opcion = 4 ENTONCES
            <acción buscar nota de un curso>
        SINO SI opcion = 5 ENTONCES
            IMPRIMIR "Saliendo del sistema..."
            SALIR DEL BUCLE
        SINO
            IMPRIMIR "Opción no válida, intente de nuevo."
        FIN_SI
    FIN_MIENTRAS

FIN
